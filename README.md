# iphaser-cli

CLI for generate a Phaser.js project. You can also use this to view documents, version 2 or version 3.

### Installation

Prerequisites: [Node.js](https://nodejs.org/en/) (>=6.x, 8.x preferred), npm version 4+ and [Git](https://git-scm.com/).

```bash
$ npm install -g iphaser-cli
```

### Usage

```bash
$ iphaser create <template-name> [project-name]
```

### Example

```bash
$ iphaser create phaser2 my-project
```

### Template list

```bash
$ iphaser list
```

You can view the latest template list with the above command

Now, you have two templates to choose from

- [phaser2](https://github.com/SleepingCat-xiaohui/iphaser-template-phaser2) - A template based on phaser2.6.2 + webpack + es6
- [phaser3](https://github.com/SleepingCat-xiaohui/iphaser-template-phaser3) - A template based on phaser3 + webpack + es6

### Docs

```bash
$ iphaser docs <version>
```

You can view the document from the above command
